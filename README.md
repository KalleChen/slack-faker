# Slack Faker

A website let you create fake slack message.

[Website](https://kallechen.github.io/slack-faker/)

![](./sample.png)

## Features
- Customized profile and time
- Multiple Quote
- Multiple and customized emoji for every quote
- Download image directly with screenshot
